题目1： jQuery 中， $(document).ready()是什么意思？

$(document).ready()方法：为防止DOM解析完成之前运行Jquery代码，若在DOM解析完成前就运行函数，操作可能失败。必须在DOM解析完成完后执行操作，可使用ready事件，作用相当于把js写到body末尾。
$(document).ready()写法：

  • $(document).redy(function(){})
  • $(function(){})
    以上两种写法是等价的

题目2： $node.html()和$node.text()的区别?

$node.html()，返回所选择元素内的html内容，包含html标签和文本内容。
$node.text()，返回所选择元素内的文本内容，不包含html标签，只包含文本内容。

题目3： $.extend 的作用和用法?

作用：将两个或更多对象的内容合并到第一个对象。
用法：jQuery.extend( [deep ], target, object1 [, objectN ] )

  • deep
    类型: Boolean
    默认情况下是false，如果是true，合并成为递归（又叫做深拷贝）。
  • target
    类型: Object
    对象扩展。这将接收新的属性。
  • object1
    类型: Object
    一个对象，它包含额外的属性合并到第一个参数。
  • objectN
    类型: Object
    包含额外的属性合并到第一个参数

题目4： jQuery 的链式调用是什么？

jQuery链式调用：在对象上一次性调动多个方法，例如：
$(this).addClass("active").siblings().removeClass("active");
因为大部分对象方法的最后是return this，所以有了链式调用，简易代码。

题目5： jQuery 中 data 函数的作用
.data() 方法允许我们在DOM元素上绑定任意类型的数据,避免了循环引用的内存泄漏风险。

  • 展示元素$node, 隐藏元素$node
    $node.show();//展示元素$node
    $node.hide();//隐藏元素$node
  • 获取元素$node 的 属性: id、src、title， 修改以上属性
    $node.attr('id');//获取元素$node 的 属性 id
    $node.attr('id','修改id值');//修改元素$node 的 id 值
    $node.attr('src');//获取元素$node 的 属性 src
    $node.attr('src','修改src值');//修改元素$node 的 src 值
    $node.attr('title');//获取元素$node 的 属性 title
    $node.attr('title','修改title值');//修改元素$node 的 title 值
  • 给$node 添加自定义属性data-src
    $node.data('data-src','自定义属性值');
  • 在$ct 内部最开头添加元素$node
    $ct.prepend($node);
  • 在$ct 内部最末尾添加元素$node
    $ct.append($node);
  • 删除$node
    $node.remove();
  • 把$ct里内容清空
    $node.empty();
  • 在$ct 里设置 html

    $ct.html('<div class="btn"></div>');

  • 获取、设置$node 的宽度、高度(分别不包括内边距、包括内边距、包括边框、包括外边距)
    $node.width();//获取$node 的宽度（不包括内边距）
    $node.height();//获取$node 的高度（不包括内边距）
    $node.width('10px');//设置$node 的宽度（不包括内边距）
    $node.height('10px');//设置$node 的高度（不包括内边距）

    $node.innerWidth();//获取$node 的宽度（包括内边距）
    $node.innerHeight();//获取$node 的高度（包括内边距）
    $node.innerWidth('10px');//设置$node 的宽度（包括内边距）
    $node.innerHeight('10px');//设置$node 的高度（包括内边距）

    $node.outerWidth();//获取$node 的宽度（包括边框）
    $node.outerHeight();//获取$node 的高度（包括边框）
    $node.outerWidth('10px');//设置$node 的宽度（包括边框）
    $node.outerHeight('10px');//设置$node 的高度（包括边框）

    $node.outerWidth(true);//获取$node 的宽度（包括边框）
    $node.outerHeight(true);//获取$node 的高度（包括边框）
    $node.outerWidth('10px',true);//设置$node 的宽度（包括外边距）
    $node.outerHeight('10px',true);//设置$node 的高度（包括外边距）
